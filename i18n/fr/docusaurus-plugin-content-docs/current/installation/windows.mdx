Attention : veuillez bien vérifier que votre applications iCloud ne provient PAS du Microsoft Store mais du site [iCloud](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe) et [iTunes](https://support.apple.com/en-us/HT210384). Avant de commencer, vérifiez d'avoir accomplis toutes les étapes dans l'onglet [Prérequis](prerequisites), et que vous possédez un [fichier d'appairage](pairing-file)!

La version non-Microsoft de iCloud a été retirée du site d'Apple vers mi-Septembre 2022. Les liens ci-dessus redirigent vers une ancienne version de iCloud qui est toujours compatible et toujours disponible directement sur les serveurs d'Apple.

1. Téléchargez et excécutez AltServer depuis [le site AltStore's](https://altstore.io/).
2. Téléchargez l'IPA SideStore que vous pouvez trouver [ici](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa).
3. Maintenez MAJ GAUCHE et cliquez sur l'icône Altserver. Appuyez sur "Sideload .ipa",et séléctionnez le fichier SideStore.ipa.
4. Approuvez "l'application de développeur" qui contient votre ID Apple et email.
    * Vous pouvez trouver cette action dans Réglages, dans "General",puis "VPN & Gestion de l'appareil".
5. Ouvrez SideStore, selectionnez votre fichier d'appairage, et connectez vous avec votre ID Apple.

Une fois que vous avez fini, suivez les [instructions post-installations](post-install).
