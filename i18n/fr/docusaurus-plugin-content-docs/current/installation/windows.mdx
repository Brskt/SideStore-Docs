Veuillez vous assurer que les versions non issues du Microsoft Store d'[iCloud](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe) et d'iTunes [(64 bits](https://apple.com/itunes.download/win64) ou [32 bits](https://apple.com/itunes.download/win32)) (iTunes devrait déjà être installé depuis la section du guide d'appairage) sont installées sur votre ordinateur. Avant de commencer, assurez-vous d'avoir suivi toutes les étapes indiquées dans la section [Prérequis](prerequisites), et d'avoir un [Fichier d'appairage](pairing-file)!

La version non-Microsoft Store d'iCloud a été retirée du site web d'Apple vers mi-septembre 2022. Le lien ci-dessus dirige vers une version plus ancienne d'iCloud qui reste compatible et est toujours disponible directement depuis les serveurs d'Apple.

1. Téléchargez `altinstaller.zip` depuis [le site d'AltStore](https://cdn.altstore.io/file/altstore/altinstaller.zip). Extrayez `altinstaller.zip` puis exécutez `setup.exe`. Dans la barre des tâches de Windows, tapez "AltServer" et cliquez sur "Exécuter en tant qu'administrateur". Assurez-vous d'autoriser l'accès aux réseaux privés si cela vous est demandé. AltStore apparaîtra comme une icône dans la barre d'état système.
2. Téléchargez le fichier IPA de SideStore que vous pouvez trouver [ici](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa)
3. Maintenez la touche Maj enfoncée et cliquez sur l'icône AltServer dans la barre d'état système. Cliquez sur "Sideload .ipa" et sélectionnez le fichier IPA de SideStore.
4. Approuvez l'"Application développeur" avec l'adresse e-mail de votre identifiant Apple.
    * Vous pouvez le trouver dans les Réglages, sous "Général", puis "VPN et gestion de l'appareil".
5. Ouvrez SideStore, sélectionnez votre fichier d'appairage et connectez-vous avec votre identifiant Apple.

Une fois terminé, suivez les [Instructions après installation](post-install).
