title: Instructions pour macOS
description: Instructions pour installer SideStore via macOS

---

Avant de commencer, assurez-vous d'avoir complété toutes les étapes indiquées dans les [Prérequis](prerequisites), et d'avoir un [Fichier d'appairage](pairing-file)!

1. Téléchargez AltServer depuis [le site d'AltStore](https://cdn.altstore.io/file/altstore/altserver.zip). Copiez AltServer.app dans votre dossier /Applications, puis lancez-le. Il devrait apparaître comme une icône dans la barre de menus.
2. Téléchargez le fichier IPA de SideStore que vous pouvez trouver [ici](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa).
3. Maintenez la touche Option enfoncée et cliquez sur l'icône AltServer dans la barre de menus. Cliquez sur "Sideload .ipa", puis sélectionnez le fichier IPA de SideStore.
4. Approuvez l'"Application développeur" avec l'adresse e-mail de votre identifiant Apple.
    * Vous pouvez le trouver dans les Réglages, sous "Général", puis "VPN et gestion des appareils".
5. Ouvrez SideStore, sélectionnez votre fichier d'appairage et connectez-vous avec votre identifiant Apple.

Une fois terminé, suivez les [Instructions après installation](post-install).
