title: Instructions pour MacOS
description: Instructions pour installer SideStore en utilisant macOS
sidebar_position: 4
---

Avant de commencer, vérifiez d'avoir accomplis toutes les étapes dans l'onglet [Prérequis](prerequisites), et que vous possédez un [fichier d'appairage](pairing-file)!

1. Téléchargez et excécutez AltServer depuis [le site AltStore's](https://altstore.io/).
2. Téléchargez l'IPA SideStore que vous pouvez trouver [ici](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa).
3. Maintenez option/alt et clicquez sur AltServer depuis la barre transversale. Clicquez sur "Sideload .ipa", et sélectionnez le fichier IPA de SideStore.
4. Approuvez "l'application de développeur" qui contient votre ID Apple et email.
    * Vous pouvez trouver cette action dans Réglages, dans "General",puis "VPN & Gestion de l'appareil".
5. Ouvrez SideStore, selectionnez votre fichier d'appairage, et connectez vous avec votre ID Apple.

Une fois que vous avez fini, suivez les [instructions post-installations](post-install).
