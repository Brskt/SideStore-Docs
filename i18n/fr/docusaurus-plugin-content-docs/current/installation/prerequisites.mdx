Le processus d'installation de SideStore peut sembler compliqué, mais avec l'utilisation ***ponctuelle*** d'un ordinateur et un peu de temps, vous aurez SideStore sur votre appareil!

## Prérequis

Pour installer SideStore, vous aurez besoin de:
- Un iPhone ou iPad [avec un code d'accès](https://support.apple.com/fr-fr/119586) fonctionnant sous iOS/iPadOS 14 ou supérieur.
- Un ordinateur sous [Windows](/docs/installation/windows.mdx), [macOS](/docs/installation/mac.mdx) ou [Linux (le support de Linux est expérimental)](/docs/installation/linux.mdx).
- Un identifiant Apple.
- L'application StosVPN.
- Un peu de temps (C'est un processus rapide, mais il vaut mieux [prévoir du temps pour le dépannage](/docs/troubleshooting/)).

Les utilisateurs d'iOS/iPadOS 16 ou supérieur *doivent* activer le mode Développeur. Pour cela:
1. Ouvrez l'application Réglages.
2. Allez dans la section 'Confidentialité et sécurité'.
3. Activez le 'Mode développeur' en bas de la page.

> **Remarque:** Vous ne verrez l'option 'Mode développeur' qu'après avoir installé SideStore avec succès.

## StosVPN

SideStore utilise un VPN intégré qui permet à SideStore de communiquer avec les services internes.

1. Téléchargez l'application StosVPN depuis l'[App Store officiel](https://apps.apple.com/fr/app/stosvpn/id6744003051).
2. Activez le VPN.

Ce VPN doit être activé à chaque fois que vous souhaitez installer ou mettre à jour des applications dans SideStore.
