## インストール

#### 指示に従いましたが、まだSideStoreをインストールできません。

### Macの手順
1. **デバイスを信頼する**: iOSデバイスをMacに接続した後、MacとiOSデバイスの両方で「信頼」を選択したことを確認してください。確認する方法は以下の通りです：
   - Finder（macOS 10.15 Catalina以降）またはiTunes（macOS 10.14 Mojave以前）を開きます。
   - デバイスを一度外して再接続し、「デバイスを信頼しますか？」というダイアログボックスが表示されるか確認してください。
2. **別のApple IDを使用する**: 別のApple IDを試してみてください。必要であれば、SideStore専用の新しいApple IDを[こちら](https://appleid.apple.com/account#!\&page=create)から無料で作成できます。

### Windowsの手順
1. **デバイスを信頼する**: iOSデバイスをWindowsコンピュータに接続した後、コンピュータとiOSデバイスの両方で「信頼」を選択したことを確認してください。確認する方法は以下の通りです：
   - iTunesを開き、「デバイスを信頼しますか？」というダイアログボックスが表示されるか確認してください。
2. **AltServerを管理者として実行する**: AltServerを右クリックして「管理者として実行」を選択し、適切な権限を確認します。
3. **別のApple IDを使用する**: 別のApple IDを試してみてください。必要であれば、SideStore専用の新しいApple IDを[こちら](https://appleid.apple.com/account#!\&page=create)から無料で作成できます。
4. **Microsoft Store版のiTunesとiCloudを使用しない**:
   - Microsoft StoreからインストールしたiTunesやiCloudはアンインストールする必要があります。
   - 以下のリンクからApple公式サイトのiTunesとiCloudの非Microsoft Store版をダウンロードしてインストールしてください：
     - [iCloudをダウンロード](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe)
     - [iTunesをダウンロード](https://www.apple.com/itunes/download/win64/)


## サインインの問題

#### SideStoreがフリーズしたり、エラーコードが表示される場合

SideStoreにサインインする際にフリーズしたりエラーコードが表示される場合、いくつかの原因が考えられます。以下の手順で問題を解決できます：

1. **Anisetteサーバーを変更する**:
   最も一般的な解決策は、別のAnisetteサーバーに切り替えることです。Anisetteサーバーがダウンしている場合もあるため、別のサーバーに切り替えることで問題が解決することがあります。手順は以下の通りです：
   - SideStoreの設定を開きます。
   - 「Anisette Servers」の設定までスクロールします。
   - リストから別のサーバーを選択します。
   - いくつかのサーバーを試して、どれが最適かを確認してください。

2. **自分のAnisetteサーバーをホストする**:
   もし自分でAnisetteサーバーを立てたい場合、公式のドキュメントで詳細な手順が提供されています。詳細は[こちら](https://docs.sidestore.io/docs/advanced/anisette)をご覧ください。

#### Apple IDでサインイン時に確認コードが届かない場合

Apple IDでSideStoreにサインインする際、確認コードが届かない場合の解決方法は以下の通りです：

**iOS 18.1未満の場合**：
1. 「設定」アプリを開きます。
2. 画面上部で自分の名前をタップします。
3. 「サインインとセキュリティ」を選択します。
4. 「二要素認証」を選び、「確認コードを取得」を選択します。

**iOS 18.1以降の場合**：
1. [iCloud](https://www.icloud.com)にブラウザでアクセスします。
2. 「サインイン」をクリックします。
3. 表示される二つのオプション（「サインイン」または「別のApple Accountを使用」）のうち、下の「別のApple Accountを使用」を選択します。
4. Apple IDとパスワードを入力します。
5. Appleから確認コードが送信されますので、それをSideStoreで使用してサインインを完了させます。


## アプリのリフレッシュまたはインストールができない場合

#### Wi-FiやVPNエラーの解決方法

1. モバイルデータを無効にし、安定したWi-Fiネットワークに接続します。
2. AppleのドメインやAnisetteサーバーのドメインが制限されていないか確認します。以下を確認してください：
   - `oscp.apple.com`などのDNSブロックを無効にする。
   - 学校や職場のWi-Fiに接続している場合は、制限のないネットワークに切り替えます。
3. StosVPNが接続されていることを確認します。
4. StosVPNを一度オフにし、再度オンにしてからSidestoreでリフレッシュを試みます。
5. 新しいペアリングファイルを作成します：
   - 上記の手順で解決しない場合、ペアリングファイルが壊れている可能性があります。新しいペアリングファイルを作成する方法については、[こちら](#)のドキュメントを参照してください。
   - 新しいペアリングファイルを作成した後、SideStoreの設定で「ペアリングファイルをリセット」を押し、新しいペアリングファイルを選択します。

## ペアリングファイルを選択できない場合

#### SideStoreでペアリングファイルを選択できない場合、以下の手順で解決できます：

1. **ファイルの拡張子を確認する**: ペアリングファイルの拡張子が`.mobiledevicepairing`または`.plist`であることを確認してください。もしそうでない場合、ペアリングファイルを圧縮せずに送信している可能性があります。その場合、ファイルが破損していることがあります。
2. **ペアリングファイルを移動する**: アプリ内でペアリングファイルを選択できない場合は、ファイル名を`ALTPairingFile.mobiledevicepairing`に変更し、SideStoreフォルダのルートディレクトリに移動してみてください。

## AltServer

#### 「AltServerが見つかりません」と表示される場合

AltServerは、AltStoreがインストールされたデバイスと同じWi-Fiネットワークに接続されているコンピュータ上で実行されている必要があります。それでもエラーが表示される場合、以下を確認してください：
- 公共のWi-Fiや職場・学校のWi-Fiを使用していませんか？その場合、Wi-Fiの設定がデバイス間の通信を制限している可能性があります。コンピュータをスマホのホットスポットに接続し、再試行してみてください。

#### Windows Store版のiCloudを使用しなければならない場合

AltStoreはAppleの公式サイトから直接iCloudをインストールする必要がありますが、Windows Store版を使用する必要がある場合は、[こちらの手順](https://www.reddit.com/r/jailbreak/comments/rp5buz/tutorial\_how\_to\_use\_altserver\_if\_you\_have\_to\_use/)に従ってください。

:::tip
もし質問が解決しない場合は、[support@altstore.io](mailto:support@altstore.io)にメールを送ってください。できるだけ早くお返事いたします。
:::

#### Windows Store版のiCloudを使用する必要がある場合はどうすればよいですか？

AltStoreは、Apple IDを認証するためにiCloudをAppleから直接インストールすることを要求します。しかし、Windows Store版を使用する必要がある場合、以下の手順に従ってください。この手順は、[こちらのreddit投稿](https://www.reddit.com/r/jailbreak/comments/rp5buz/tutorial\_how\_to\_use\_altserver\_if\_you\_have\_to\_use/?utm\_source=share\&utm\_medium=ios\_app\&utm\_name=iossmf)からコピーしたものです：

1. 公式のAppleウェブサイトから[iTunes](https://www.apple.com/itunes/)をダウンロードしてインストールします。**必ずスクロールダウン**して、「Looking for other versions?」の横にある「Windows」をクリックしてください。
2. 公式のAppleウェブサイトから[iCloud](https://support.apple.com/en-us/HT204283)をダウンロードしてインストールします。**必ずスクロールダウン**し、次のテキストのリンクをクリックしてください：「* Windows 7およびWindows 8では、_iCloud for WindowsをAppleのウェブサイトからダウンロード_できます」。
3. 「C:\Program Files (x86)\Common Files\Apple」フォルダを探し、「Apple Application Support」と「Internet Services」フォルダをコピーします。その後、新しいフォルダを作成し、好きな名前を付けてその中に貼り付けます。
4. コンピュータからiCloudを削除します。**絶対に**iTunesは削除しないでください！（また、iTunesやiCloudと一緒にインストールされた「Apple Mobile Device Support」や「Apple Application Support」を削除しないでください。）
5. Windows Storeから[iCloud](https://www.microsoft.com/store/productId/9PKTQ5699M62)をインストールします。
6. [AltServer](https://altstore.io/)をダウンロードしてインストールします。初めて開いたとき、「Choose Folder...」をクリックし、3番目の手順で作成したフォルダを選択してください。
