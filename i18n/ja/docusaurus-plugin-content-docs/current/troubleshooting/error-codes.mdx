## SideStore エラー

#### (1000) An unknown error occurred.

`AltStore.OperationError`

予期しないエラーが発生したことを意味します。[トラブルシューティングガイド](./)を確認し、問題が解決しない場合は直接お問い合わせください。

#### (1001) The operation returned an unknown result.

予期しないエラーが発生したことを意味します。[トラブルシューティングガイド](./)を確認し、問題が解決しない場合は直接お問い合わせください。

#### (1002) The operation was cancelled.

`AltStore.OperationError`

実行した操作がキャンセルされたことを意味します。再試行してください。

#### (1003) The operation timed out.

`AltStore.OperationError`

要求された操作を完了するのに問題が発生したことを意味します。再試行し、追加のサポートが必要な場合は[トラブルシューティングガイド](./)を確認してください。問題が解決しない場合は直接お問い合わせください。

#### (1004) You are not signed in.

`AltStore.OperationError`

Apple ID情報を設定タブに入力する必要があることを意味します。

#### (1005) The app could not be found.

`AltStore.OperationError`

指定されたURLでファイルが見つからなかったことを意味します。

#### (1006) AltStore could not determine this device's UDID.

`AltStore.OperationError`

SideStoreがAltServerによってインストールされたデバイスのUDIDを読み取れなかったことを意味します。最新のAltServerバージョンを使用してSideStoreをインストールしてください。

#### (1007) This app is in an invalid format.

`AltStore.OperationError`

インストールしようとしているアプリが標準の.ipaファイルではないことを意味します。別の場所からアプリをダウンロードしてみてください。

#### (1008) Invalid parameters.

`AltStore.OperationError`

内部エラーが発生し、SideStoreが必要な情報を提供できなかったことを意味します。再試行し、[トラブルシューティングガイド](./)を確認してください。問題が解決しない場合は直接お問い合わせください。

#### (1009) You cannot register more than 10 App IDs within a 7-day period.

`AltStore.OperationError`

7日間に10個以上のApp IDを登録できないことを意味します。My AppsタブでApp IDの有効期限を確認してください。

#### (1010) There are no AltStore sources.

`AltStore.OperationError`

SideStoreが追加したソースの変更を取得できない内部エラーが発生したことを意味します。再試行し、[トラブルシューティングガイド](./)を確認してください。問題が解決しない場合は直接お問い合わせください。

#### (1011) AltStore was denied permission to launch the app.

`AltStore.OperationError`

「AltStoreが\[アプリ名\]を開こうとしています」というアラートで「キャンセル」をタップしたか、Apple IDを設定で信頼する必要があることを意味します。

#### (1012) AltStore's shared app group could not be accessed.

`AltStore.OperationError`

アプリのバックアップが保存されているディレクトリにアクセスできなかったことを意味します。最新のAltServerバージョンを使用してSideStoreをインストールしてください。

#### (1200) AltServer could not be found.

`AltStore.OperationError`

SideStoreがローカルネットワーク上でAltServerを発見できないことを意味します。PC/MacでAltServerが実行されているのと同じWi-Fiネットワークに接続していることを確認してください。または、USB経由でデバイスをPCに接続してみてください。

#### (1201) A connection to AltServer could not be established.

`AltStore.OperationError`

PC/MacのAltServerに接続できなかったことを意味します。Wi-Fi同期が有効であることを確認するか、デバイスをUSBで直接接続してみてください。

#### (1202) The connection to AltServer was dropped.

`AltStore.OperationError`

PC/MacのAltServerとの接続が切断されたことを意味します。Wi-Fi同期が有効であることを確認するか、デバイスをUSBで直接接続してみてください。

## AltServer / SideServer エラー

#### (0) Exception Error

`AltServer.ExceptionError`

多くの理由でこのエラーが発生する可能性があります。以下にいくつかの例を示します（これは完全なリストではありません）。

"vector too long"

Windows Defenderがリクエストをブロックしていることを意味します。SideStoreを正常に動作させるために、まずWindowsのリアルタイム保護を無効にする必要があるかもしれません。

#### (1999) Underlying Error

`AltServer.ServerError`

AltServerで不明なエラーが発生したことを意味します。問題が解決しない場合は、サポートチャンネルに連絡することをおすすめします。

#### (2000) Unknown Error

`AltServer.ServerError`

AltServerで不明なエラーが発生したことを意味します。問題が解決しない場合は、サポートチャンネルに連絡することをおすすめします。

#### (2001) Could not connect to AltServer. / There was an error connecting to the device.

`AltServer.ServerError`

AltServerがiOSデバイスへの接続に問題があることを意味します。まだ接続していない場合は、デバイスをPCに直接接続してみてください。

問題が解決しない場合は、[トラブルシューティングガイド](./)を参照してください。

#### (2002) Lost Connection to AltServer.

`AltServer.ServerError`

AltServerとiOSデバイスとの接続が切断されたことを意味します。デバイスとPCが同じWi-Fiネットワークに接続されていることを確認し、再試行してください。

問題が解決しない場合は、[トラブルシューティングガイド](./)を参照してください。

#### (2003) AltServer could not find this device.

`AltServer.ServerError`

AltServerがUSBまたはWi-Fi経由でiOSデバイスを見つけられなかったことを意味します。デバイスとPCが同じWi-Fiネットワークに接続されていることを確認し、デバイスが信頼されていることを確認して、再試行してください。

問題が解決しない場合は、[トラブルシューティングガイド](./)を参照してください。

#### (2004) Failed to write app data to device.

`AltServer.ServerError`

AltServerがデバイスにファイルを書き込めなかった、または中断されたことを意味します。デバイスとPCが同じWi-Fiネットワークに接続されていることを確認し、デバイスが信頼されていることを確認して、再試行してください。

問題が解決しない場合は、[トラブルシューティングガイド](./)を参照してください。

#### (2005) AltServer received an invalid request.

`AltServer.ServerError`

AltServerが理解できないリクエストを受け取ったことを意味します。古いAltServerバージョンを使用している可能性があるため、最新バージョンにアップデートしてください。

#### (2006) AltServer sent an invalid response.

`AltServer.ServerError`

AltServerからSideStoreが理解できないレスポンスを受け取ったことを意味します。古いSideStoreバージョンを使用している可能性があるため、最新バージョンにアップデートしてください。

#### (2007) The app is invalid.

`AltServer.ServerError`

インストールしようとしたアプリが無効な形式であることを意味します。別のソースからダウンロードしてみてください。

#### (2008) An error occurred while installing the app.

`AltServer.ServerError`

iOSがインストールしようとしたアプリを拒否したことを意味します。iOSバージョンに互換性のあるバージョンを使用してください。

#### (2009) Cannot activate more than 3 apps with a non-developer Apple ID.

`AltServer.ServerError`

無料の開発者アカウントでサイドロードできるアプリの最大数に達したことを意味します。この制限を解除するには、[developer.apple.com](https://developer.apple.com)で有料開発者アカウントに登録する必要があります。

#### (2010) Your device must be running iOS 12.2 or later to install AltStore.

`AltServer.ServerError`

AltStoreを使用するためには、最低でもiOS 12.2以降にアップデートする必要があることを意味します。

#### (2011) AltServer does not support this request.

`AltServer.ServerError`

AltServerが認識できないリクエストを受け取ったことを意味します。古いAltServerバージョンを使用している可能性があるため、最新バージョンにアップデートしてください。

#### (2012) Received an unknown response from AltServer.

`AltServer.ServerError`

AltServerからAltStoreが認識できないレスポンスを受け取ったことを意味します。古いSideStoreバージョンを使用している可能性があるため、最新バージョンにアップデートしてください。

#### (2013) The provided anisette data is invalid.

`AltServer.ServerError`

AltStoreが使用したApple ID情報が無効であるか、期限切れであることを意味します。Windowsで最新バージョンのiCloudを使用しており、日付と時刻が正しいことを確認してください。

#### (2014) AltServer could not connect to Mail plug-in. (macOS)

`AltServer.ServerError`

AltServerがメールプラグインを見つけられなかったか、メールアプリがMacで実行されていないことを意味します。メールアプリが実行されていることを確認し、プラグインが有効になっていることを確認してください。

#### (2015) Could not find provisioning profile.

`AltServer.ServerError`

このアプリをインストールするために必要なプロビジョニングプロファイルが見つからなかったことを意味します。AltStore自体に問題がある可能性があるため、サポートまたはPatreonの問い合わせ先に連絡してください。

#### (2016) An error occurred while removing the app.

`AltServer.ServerError`

アプリを削除する際に問題が発生したことを意味します。エラーログを確認して、問題の詳細を確認してください。

#### (2100) The Requested app is not currently running on the device.

`AltServer.ServerError`

要求されたアプリがデバイスで実行されていない状態でJITを有効にしようとしたことを意味します。アプリを起動して実行してからJITを有効にしてください。

#### (2101) The disk is incompatible with [iOS version].

`AltServer.ServerError`

JITを有効にするために必要な開発者ディスクがデバイスのOSバージョンと互換性がないことを意味します。最新のAltStoreとAltServerを実行しており、後で再試行してください。

## Apple API エラー

#### (1100) Your session has expired.

`Apple.APIError`

Apple IDの認証情報が期限切れまたは無効であることを意味します。コンピュータの日付と時刻が正確であることを確認し、再試行してください。

#### (1102) Apple ID cannot be used for development.

`Apple.APIError`

Appleの利用規約に同意する必要があるかもしれません。[developer.apple.com](https://developer.apple.com)で確認してください。それでも解決しない場合は、サイドローディング専用の新しいApple IDを作成してください。

#### (-20101) Your account information was entered incorrectly.

`Apple.APIError`

このエラーは、Apple IDの認証に使用されたユーザー名またはパスワードが期限切れか無効であることを意味します。両方の入力が正しいか、コンピュータの日付と時刻が正確であるか確認してください。

## Windows エラー

#### (1100) Your session has expired. Please log in.

`NSURLErrorDomain`

1. iTunesが閉じていることを確認してください。
2. Program Data -> Apple Computer -> iTunes -> ADI に移動し、ファイルを削除します。
3. iTunesを起動します。
4. SideStoreのインストールを再試行してください。

## その他のエラー

#### (512) Failed to write to disk

`NSCocoaErrorDomain`

このエラーは、ディスクへの書き込み中にエラーが発生したことを意味します。スマートフォンとPCの両方に十分なディスクスペースがあることを確認し、再試行してください。

#### (1) AltJIT failed with error code 1 (iOS 17)

`AltServer.ProcessError`

iOS 17デバイスでJITを有効にしようとしている場合、このエラーが表示されることがあります。すべての依存関係をインストールし、アプリを開いたままでデバイスを接続したままJITの有効化が完了するまで待ってください。
