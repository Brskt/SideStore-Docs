## SideStoreに関するよくある問題

### AFC接続の失敗
#### 原因
接続の問題、または無効なペアリングファイル。
#### 解決策
1. 再接続を試みる
2. JitterbugPairを使用して新しいペアリングファイルを生成し、設定メニューからインポートする。[ペアリングファイルの手順](/docs/installation/pairing-file)を参照

### WiFiまたはVPN接続の不具合
#### 原因
minimuxerサービスとの接続の問題。
#### 解決策
1. WireGuardを使用している場合、StosVPNに切り替える
2. 再度試してみる（ボタンを何度も押すと効果的な場合があります）
3. SideStoreを再起動する
4. StosVPNを再起動する
5. JitterbugPairを使用して新しいペアリングファイルを生成し、設定メニューからインポートする。[ペアリングファイルの手順](/docs/installation/pairing-file)を参照

### SideStoreがインストール途中でフリーズする
#### 原因
キャッシュの問題。
#### 解決策
1. SideStoreを再起動する
2. キャッシュをクリアする
3. Anisetteサーバーを変更する
4. adi.pbをリセットする
5. SideStoreからサインアウトし、再度サインインする
6. WireGuardを使用する
7. デバイスを再起動する
8. ペアリングファイルを再生成する
9. SideStoreを再インストールする

### SideStoreを起動できない
#### 原因
複数の原因が考えられる。
#### 解決策
1. SideStoreをAltserverを通じてインストールしたことを確認する。
2. SideStoreを再インストールする

上記がうまくいかない場合、以下の手順を試してみてください：

1. SideStoreと、SideStoreからインストールしたすべてのアプリを削除する。
2. SideStoreを再インストールする。
3. ペアリングファイルを選択し、ログインする。（SideStoreの更新を促すポップアップが表示されますが、「いいえ」を選択）
4. Sidestore.ipaファイルをデバイスにインストールする。
5. Sidestore.ipaファイルを通常のアプリインストールと同様にSideStoreにインポートする。
6. これにより、SideStoreがクリーンな状態で更新され、問題が解決する可能性があります。
