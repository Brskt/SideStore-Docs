# よくある質問（FAQ）

## アプリのインストール時に「開発元が信頼されていません」と表示されます

アプリを開く前に、[設定] → [一般] → [VPNとデバイス管理] に移動して、SideStoreのプロファイルを「信頼」する必要があります。

## アプリが「インストール中」のまま進みません

- まず、アプリがホーム画面に表示されていることを確認してください。
- その状態でSideStoreアプリを開き、更新が完了するまで待ちます。

## アプリが勝手に削除されました

Appleは署名の有効期限が切れたエンタープライズアプリや開発者アプリを自動的に削除することがあります。  
SideStoreを定期的に開いて署名を更新するようにしてください。

## デバイスにSideStoreが表示されません

- 同じWi-Fiネットワーク上にAltServerがあることを確認してください。
- SideStoreアプリ内で「ペアリングされたデバイスを再スキャン」してください。
- 接続がうまくいかない場合は、AltServerとPCを再起動してみてください。

## 「無効なアプリの署名」または「アプリを検証できません」と表示されます

このエラーは、署名が無効または期限切れになっていることを意味します。SideStoreアプリを開いてアプリの署名を再度行ってください。

## Apple IDとパスワードの入力が求められます。安全ですか？

SideStoreはあなたのApple IDを使用してアプリに署名します。**この情報はローカルでのみ使用され、どこにも送信されません。**  
ただし、心配であればサイドローディング専用のApple IDを作成することをおすすめします。

## サイドローディングできるアプリの数に制限はありますか？

Appleの制限により、**最大で3つのアプリ**しか同時にインストールできません（アプリごとに1つの署名枠が必要です）。

## SideStoreを完全にアンインストールするには？

1. SideStoreアプリを削除します。
2. 設定アプリからSideStoreのプロファイルを削除します。
3. 必要であれば、AltServerからペアリングを解除してください。
