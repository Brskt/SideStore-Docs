SideStoreをインストールするための最後のステップは、SideStore自体をリフレッシュすることです。

これを行うには、次の手順を実行してください：

1. StosVPNに接続します。
2. SideStoreを開きます。
3. まだ選択していない場合は、ペアリングファイルを選択します。
4. 「マイアプリ」に移動します。
5. 「x日後」と表示されているボタンを選択します。ここで「x」は、期限が切れるまでの残り日数です。
    * 成功すると、Safariの空白タブが開くか、ホーム画面に戻ります。また、これを行うよう通知が届きます。

もしSideStoreがリフレッシュされない場合、早期期限切れを引き起こすバグが発生することがあります。その場合、SideStoreを再インストールする必要があります。この作業を自動化する計画はありますが、現時点では手動で行う必要があります。
