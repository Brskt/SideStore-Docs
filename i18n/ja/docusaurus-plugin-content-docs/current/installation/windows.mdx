Windowsをお使いの場合は、**Microsoft Store版ではない**以下のソフトがインストールされていることを確認してください：  
- [iCloud](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe)  
- [iTunes](https://support.apple.com/en-us/HT210384)  

また、事前に[前提条件](prerequisites) と [ペアリングファイル](pairing-file) の準備を完了していることを確認してください。

> **注意:** Microsoft Store版のiCloudは2022年9月中旬頃にAppleのサイトから削除されました。上記リンクは互換性のある古いバージョンへ直接アクセスできます。

1. [AltStoreのサイト](https://altstore.io/)からAltServerをダウンロードして実行します。  
2. 最新のSideStore IPAを[こちら](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa)からダウンロードします。  
3. Shiftキーを押しながらタスクトレイのAltServerアイコンをクリックし、「Sideload .ipa」を選択してSideStore IPAを指定します。  
4. Apple IDのメールアドレスで「Developer App」を承認します。  
   * 設定アプリの「一般」→「VPNとデバイス管理」から確認できます。
5. SideStoreを開き、ペアリングファイルを選択してからApple IDでサインインします。

インストール完了後は、[インストール後の手順](post-install) に進んでください。
