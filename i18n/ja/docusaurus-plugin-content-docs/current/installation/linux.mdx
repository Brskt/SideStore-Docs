LinuxでSideStore（または他のIPA）をインストールするには、Dockerがインストールされていることを確認してください。開始する前に、[前提条件](prerequisites)とStosVPNのインストールを完了していることを確認してください。

1. Dockerでaltconを実行します
```bash
docker run --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```
Podmanを使用してaltconを実行することもできます
```bash
podman run --privileged --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```
Altconはデバイスとのペアリングを開始します。まだ接続していない場合は、デバイスを接続してください。「信頼」をタップし、パスコードを入力します。

完了したら、[インストール後の手順](post-install)に進んでください。
